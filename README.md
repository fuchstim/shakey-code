# ShakeyCode
### ShakeyCode allows you to generate and scan QR-Like codes using Shakey as a motive.

Demo: https://fuchstim.github.io/shakey-code

## Installation
```
npm install
```

### Run Development Server
```
npm run serve
```

### Compile Production Build
```
npm run build
```

### Lint and Fix Files
```
npm run lint
```
